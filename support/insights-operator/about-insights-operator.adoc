= About Insight Operator
include::modules/common-attributes.adoc[]
:context: insights-operator

toc::[]

Insights-operator collects anonymized information about:

* The version of the {product-title} cluster and it's components
* The health, status and current errors reported by {product-title} components and nodes
* Network topology information
* The features in use

This information is used by Red Hat to be able to provide better support for the {product-title} cluster. None of the information is shared with third parties.
